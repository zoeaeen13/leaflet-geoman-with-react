## React：GPS coordinates with OSM + Leaflet 
### [Live Demo](https://zoeaeen13.github.io/leaflet-geoman-with-react/)

![](https://i.imgur.com/HIr2bF5.jpg)

### 功能
- 以多邊形圈選所需範圍
- 支援放大縮小
- 可編輯及刪除已圈選範圍
- 一鍵生成座標點位（目前僅支援一區域）
- 複製座標
